# Comparative Induction

Comparison reveals why choices change. It is not enough to count repeated surface features.

## Corpus Contrasts

| Comparison | What it can expose |
|---|---|
| Same creator across works | recurring grammar, signature, possible doctrine |
| Same work before and after revision | decision criteria and priority direction |
| Same brief by different creators | genre convention versus creator-specific choice |
| Accepted versus rejected alternatives | taste boundaries and anti-goals |
| Successful versus failed outputs | validity conditions and failure boundaries |
| Same system across platforms | invariant function versus delivery adapter |

## Alignment Record

Align artifacts by production function and context, not by section number or object name. Record:

- source, creator/system, brief, audience, medium, objective, and outcome
- opportunity: which decision could have been made
- observed choice and plausible alternatives
- function of the choice
- revision or acceptance status
- context variable that changed
- evidence location and confidence

Use `assets/templates/comparison-matrix.csv`.

## Induction Signals

- Present across eligible cases: invariant candidate.
- Changes consistently with a context variable: conditional rule.
- Repeatedly differs from genre peers: signature hypothesis.
- Repeatedly removed or rejected: anti-goal or lower-priority value.
- Revision repeatedly moves in one direction: value-priority hypothesis.
- Absent only where there was a real opportunity to use it: meaningful negative evidence.

Absence is not an anti-goal when the artifact never had an opportunity to include the element.

## Hypothesis Record

Every induced principle needs:

- claim
- supporting cases and source locations
- contradictory or counterexample cases
- eligible-case denominator
- context scope
- confidence and basis
- a prediction for a held-out case
- promotion requirement

## Prediction Test

Before reading the held-out creator choice, use the hypothesis to predict:

1. which option will be selected
2. which value wins if options conflict
3. what will be removed or preserved during revision

A doctrine hypothesis that cannot improve prediction is descriptive branding, not recovered production knowledge.
